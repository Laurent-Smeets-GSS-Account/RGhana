R Markdown

What is it * R Markdown is a file format for making dynamic documents

Who is it for * For communicating to decision makers, who want to focus on the conclusions, not the code behind the analysis.

  • For collaborating with other data scientists (including future you!), who are interested in both your conclusions, and how you reached them (i.e. the code).

  • As an environment in which to do data science, as a modern day lab notebook where you can capture not only what you did, but also what you were thinking.

R Markdown code

What does R Markdown contains

It contains three important types of content:

  • An (optional) YAML header surrounded by —s.
  • Chunks of R code surrounded by ```.
  • Text mixed with simple text formatting like # heading and italics.
